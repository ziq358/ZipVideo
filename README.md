# ZipVideo
视频 悬浮框
点击 按钮上方的空白位置，播放视频， 点击按钮显示悬浮框
按钮下方是 一个控制器，实现的功能有： 播放，暂停，全屏， 左部分上下滑动调节亮度， 右部分上下滑动调节音量
